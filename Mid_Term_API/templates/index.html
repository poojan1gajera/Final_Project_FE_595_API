They fight crime NLP API Documentation

by Rohan, Monica, Ishani, Poojan.

Documentation for the Final Group Project: NLP API.

This is a course project for FE-595. This API takes in a CSV file which can be submitted on the top right corner of the home page. The primary aim of this project is to get sentiment analysis of the Client-Reprenstative chat for an event at a venue. On submitting the POST request gets in the data and the backend sentiment analysis runs on a python file. The final output is shown on a new page.

Feel free to play around with this project API.
